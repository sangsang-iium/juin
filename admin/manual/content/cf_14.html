<팝업 생성 및 수정>
[제목] : 팝업창 타이틀에 실제 노출되는 팝업 제목입니다.
[팝업크기(pixel)] : 가로크기 * 세로크기 순으로 입력하세요!
[팝업위치(pixel)] : 인터넷창 상단위치 * 좌측위치 순으로 입력하세요!
[실행기간] : 팝업 실행 시작일 * 종료일 순으로 입력하세요!
[실행]
　 ※ 바로실행 : 팝업이 실행 됩니다.
　 ※ 실행안함 : 팝업이 실행 되지 않습니다.
[실행사이트]
　 ※ 쇼핑몰 : 쇼핑몰에서만 실행 됩니다. 회사홈페이지에서는 실행되지 않습니다.
　 ※ 회사홈페이지 : 회사홈페이지에서만 실행 됩니다. 쇼핑몰에서는 실행되지 않습니다.

① [오늘은 더이상 열지 않음]에 체크하시면 팝업창이 닫히며 클릭하는 시간부터 24시간 동안 다시 실행 되지 않습니다.
② [닫기] 버튼을 클릭하시면 팝업창이 닫히지만 새로고침 할 경우 다시 재실행 됩니다.

<참고사항>
※ 본사에서 등록한 팝업은 분양몰에서는 실행 되지 않습니다.
※ 즉! 가맹점, 본사 모두 본인이 등록한 팝업만 본인 사이트에서 실행 됩니다.
