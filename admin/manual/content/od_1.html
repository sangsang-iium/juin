<주문관리 공통>
  • [오늘 접수된주문] : 금일 주문된 내역만 노출 됩니다.
  • [1단계_주문확인] : 무통장입금 및 가상계좌로 결제시 노출 됩니다. (결제대기 주문건)
  • [2단계_상품준비중] : 결제완료된 주문건이 노출 됩니다. 신용카드,계좌이체,핸드폰결제시
  • [3단계_배송대기] : 주문상품이 출고 전 상태입니다.
  • [4단계_배송중] : 주문상품을 출고 후 상태입니다.
  • [5단계_배송완료] : 배송이 완료 된 주문상품이 노출 됩니다.
  • [주문취소 요청] : 고객이 주문취소 요청한 내역이 노출 됩니다.
  • [입금후_주문취소] : 신용카드,계좌이체,핸드폰결제 후 취소한 내역이 노출 됩니다.
  • [입금전_주문취소] : 무통장입금 및 가상계좌로 주문결제 후 취소한 내역이 노출 됩니다.
  • [상품 반품목록] : 배송이 완료된 주문건중 반품처리된 내역이 노출 됩니다.
  • [상품 교환목록] : 배송이 완료된 주문건중 교환처리된 내역이 노출 됩니다.
  • [일련번호] : 상품주문시 개별적으로 중복되지않는 고유값 입니다.
  • [주문번호] : 여러상품을 복수구매를 하더라도 주문번호는 동일하게 적용됩니다.
  • [회원] : 회원이 주문시 "√"표시가 노출 됩니다.
  • 　  PC버전에서 주문시 노출되는 아이콘입니다.
  • 　  모바일쇼핑몰에서 주문시 노출되는 아이콘입니다.
  • [결제방법] : 신용카드,무통장,계좌이체,핸드폰,가상계좌,에스크로 계좌이체,에스크로 가상계좌중 하나가 노출 됩니다.
  • [가맹점] : 판매수수료를 지급받을 가맹점 아이디가 출력됩니다. 배송완료시 판매수수료가 적립됩니다.
  • [판매자] : 실제 상품을 발주할 회원정보입니다.
