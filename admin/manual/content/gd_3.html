<상품 가격정책>
1, [판매마진]
　 ※ 시중가의 : ○○%가 되도록 적용하기 예시 : 시중가가 10,000원 일경우 10%로 변경시 11,000원이 됩니다.
　 ※ 매입가의 : ○○%가 되도록 적용하기 예시 : 도매가가 8,000원 일경우 10%로 변경시 8,800원이 됩니다.
　 ※ 공급가의 : ○○%가 되도록 적용하기 예시 : 공급가가 8,000원 일경우 10%로 변경시 8,800원이 됩니다.
2, [판매가격]
　 ※ 시중가의 : ○○%가 되도록 적용하기 예시 : 시중가가 10,000원 일경우 90%로 변경시 9,000원이 됩니다.
　 ※ 매입가의 : ○○%가 되도록 적용하기 예시 : 도매가가 8,000원 일경우 110%로 변경시 8,800원이 됩니다.
　 ※ 공급가의 : ○○%가 되도록 적용하기 예시 : 공급가가 8,000원 일경우 110%로 변경시 8,800원이 됩니다.
3, [적립금액]
　 ※ 시중가의 : ○○%가 되도록 적용하기 예시 : 시중가가 10,000원 일경우 10%로 변경시 1,000원이 됩니다.
　 ※ 매입가의 : ○○%가 되도록 적용하기 예시 : 도매가가 8,000원 일경우 10%로 변경시 800원이 됩니다.
　 ※ 공급가의 : ○○%가 되도록 적용하기 예시 : 공급가가 8,000원 일경우 10%로 변경시 800원이 됩니다.
