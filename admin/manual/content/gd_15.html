<쿠폰관리 (인쇄용)>
  • ② [인증번호] : 생성 된 쿠폰 리스트 항목을 상세히 보실 수 있습니다.
  • ③ [수정] : 버튼을 클릭시 "수정" 모드로 접속 합니다.
  • ④ [삭제] : 버튼을 클릭시 개별적으로 삭제 합니다.
  • ⑤ [선택삭제] : 삭제 하실 항목을 ①번 항목에 체크 후 "선택삭제" 버튼을 클릭합니다.
  • ⑥ [추가하기] : 신규 쿠폰을 생성합니다.
<보충설명>
  • 1) 일련번호 : 쿠폰생성시 그룹번호 (쿠폰번호 아님)
    2) 시작 : 쿠폰 유효 시작일
    3) 종료 : 쿠폰 유효 종료일
    4) 금액 : 쿠폰 정상인증시 포인트로 전환될 금액
    5) 매수 : 쿠폰생성 총 매수
    6) 사용 : 인증완료 된 쿠폰 수
    7) 발행금액 : 총 쿠폰발행 금액 (금액 X 매수 = 발행금액)
    8) 사용금액 : 인증완료 후 포인트로 전환된 쿠폰 총 금액
<고객 쿠폰인증 절차>
  • ※ 포인트로 전환이 완료되면 상품주문시 즉시 사용 가능합니다.
