<모듈편집 & 생성하기>
  • [스킨 신규생성 절차]
    - 상단 스킨목록중에 생성/수정할 항목을 클릭 후 ①번 스킨을 선택 합니다.
    - ②번 새로작성을 클릭합니다.
    - ③번 스킨명을 입력 합니다. 영문자, 숫자, _ 만 가능 (공백없이 50자 이내, 한글입력 불가!!)
    - 마지막으로 하단에 저장 버튼을 클릭하시면 스킨이 생성 됩니다.
  • [스킨 HTML수정 절차]
    - 상단 스킨목록중에 생성/수정할 항목을 클릭 후 ①번 스킨을 선택 합니다.
    - HTML 소스수정이 완료된 후 ④번 덮어쓰기를 클릭합니다.
    - 마지막으로 하단에 저장 버튼을 클릭하시면 스킨이 수정 됩니다.
  • [스킨 삭제 절차]
    - 상단 스킨목록중에 생성/수정할 항목을 클릭 후 ①번 스킨을 선택 합니다.
    - ⑤번 삭제를 클릭합니다.
    - 마지막으로 하단에 저장 버튼을 클릭하시면 스킨이 삭제 됩니다. (기본 스킨은 삭제하지 마세요!)
    <템플릿 생성 절차안내>
  • ※ 레이아웃은 크게 4가지로 분류 됩니다.
    메인페이지, 서브페이지, 상품상세페이지, 그외 기타페이지 (고객센터, 회원가입, 주문페이지등)
  • ※ 작업 우선순위 안내
    절차 1) 개별 스킨관리 > 모듈편집 & 생성하기
    절차 2) 레이아웃 관리 > ① 메인페이지설정, ② 서브페이지설정, ③ 상품상세보기설정, ④ 기타페이지설정
    절차 3) 템플릿 관리 > ① 템플릿 생성하기, ② 템플릿 목록보기(대표 템플릿 선택)
  • ※ 사용방법은 메뉴얼 해당 페이지를 참조 하시기 바랍니다.
