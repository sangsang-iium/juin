<상점 배너관리>
  • ② [배너수정] : 버튼을 클릭시 "배너수정" 모드로 접속 합니다.
  • ③ [배너삭제] : 버튼을 클릭시 개별적으로 배너를 삭제 합니다.
  • ④ [선택수정] : CODE, 감춤, 사이즈, 링크주소등 수정하실 항목에 값을 변경하시고 ①번 항목에 체크 후 "선택수정" 버튼을 클릭합니다.
  • ⑤ [선택삭제] : 삭제 하실 배너를 ①번 항목에 체크 후 "선택삭제" 버튼을 클릭합니다.
  • ⑥ [추가하기] : 신규배너를 등록 합니다
<배너 신규등록>
  • ② [적용대상] : 적용대상 회원ID를 먼저 선택을 하세요! 기본값은 "admin" 입니다.
  • ③ [배너파일] : GIF, JPG, PNG 파일만 등록 가능합니다.
  • ④ [링크주소] : 쇼핑몰 내부 링크일 경우 http://도메인은 입력하시면 안됩니다. 하단 예시 참조!
    쇼핑몰 링크 예시) /shop/view.php?index_no=1
    모바일 링크 예시) /m/tb/view.php?gs_id=1
  • ⑤ [링크유형] : "현재창, 새창" 중 선택하세요.
  • ⑥ [배너크기] : ⑦번 항목 배너 코드를 선택하시면 가로, 세로 사이즈 값이 자동으로 입력 됩니다. 개별입력 가능!
  • ⑦ [배너코드] : ①번 항목 배너 테이블 표를 참조하신 후 선택하세요!
  • ⑧ [출력여부] : 감춤에 체크하시면 배너가 노출되지 않습니다.
    <참고사항>
  • ※ 가맹점 회원은 본인의 마이페이지에서 배너등록이 가능합니다.
  • ※ 가맹점이 개별적으로 등록할 수 없는 상황일때 관리자가 강제로 등록해 주시면 됩니다.
  • ※ 가맹점이 배너등록을 하지 않을 경우 본사 배너가 기본으로 노출 됩니다.
