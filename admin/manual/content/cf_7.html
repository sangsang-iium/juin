※ 신용카드, 계좌이체, 가상계좌, 휴대폰결제, 현금영수증은 KCP와 계약이 선행되야 합니다.
1, 결제수단 설정 : KCP와 계약한 결제수단을 설정하시면 됩니다.
- KCP와 계약을 신청한 이후에 KCP에서 메일로 고객님께 SITE CODE와 SITE KEY 를 보내드립니다.
 2, KCP PG ID : 발급받으신 KCP PG ID를 입력하세요.
3, KCP KEY : 발급받으신 KCP KEY를 입력하세요.
4, KCP PG ID 모바일웹 : 발급받으신 모바일웹 KCP PG ID를 입력하세요.
5, KCP KEY 모바일웹 : 발급받으신 모바일웹 KCP KEY를 입력하세요.
6, 현금영수증 발급
- 주의1 : 현금영수증 사용 시 KCP 상점관리자 페이지에서 현금영수증 사용 동의를 하셔야 합니다.
- 주의2 : 현금영수증 발급 취소는 PG사에서 지원하는 현금영수증 취소 기능을 사용하시기 바랍니다.
 7, 최대 할부 개월
- 할부옵션 : Payplus Plug-in에서 카드결제시 최대로 표시할 할부개월 수를 설정합니다. (0 ~ 18 까지 설정 가능)
- 주의 : 할부 선택은 결제금액이 50,000원 이상일 경우에만 가능, 50000원 미만의 금액은 일시불로만 표기됩니다
- 예) 값을 "5" 로 설정했을 경우 => 카드결제시 결제창에 일시불부터 5개월까지 선택가능
 8, 무이자 사용 여부 : 무이자의 사용여부를 설정합니다.
- 무이자결제 사용시 KCP와 계약 체결 되어야 합니다
- 참고 : 일반할부 - KCP 이벤트 이외에 설정 된 모든 무이자 설정을 무시합니다
 9, 무이자 할부 기간
- 1) 전 카드사 2,3,4개월 무이자 설정 시 : 예) ALL-02:03:04
- 2) BC 2,3,6개월, 국민 3,6개월, 삼성 6,9개월 무이자 설정 시 : 예) CCBC-02:03:06,CCKM-03:06,CCSS-03:06:04
 10, 가상계좌 입금통보 URL : http://대표도메인/shop/kcp/kcp_common.php
- KCP 관리자 > 상점정보관리 > 정보변경 > 공통URL 정보 > 공통URL 변경후자동으로 입금 통보됩니다.
11, 에스크로 사용여부 : 에스크로를 계약하셨다면 사용으로 체크하세요!
12, Escrow 결제 수단 : 계좌이체 또는 가상계좌 결제수단 사용 시 자동으로 적용됩니다. 단! 소비자회원이 재선택 가능!

<무이자결제>
할부판매에 대한 이자는 구매자가 부담하는 것이 할부거래의 원칙입니다.
하지만 구매자 대신 상점에서 할부 이자를 부담하고 할부 판매를 할 수도 있는데,
무이자할부 판매를 시행하시려면 KCP와의 별도 계약이 필요하며,
결제 페이지에서도 이에 대한 설정을 해주어야 합니다.
주의)무이자결제를 체크하고, 무이자기간을 아무것도 기재하지 않을경우 : 카드에 따라 전 할부기간 무이자가 적용되어 판매자가 수수료를 부담해야 하니 꼭 주의하시기 바랍니다.
