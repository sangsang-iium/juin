<사업자정보>
1, 사업자정보는 쇼핑몰에서는 노출 되지 않으며 모바일 쇼핑몰 하단에 노출 됩니다.
2, 가맹점은 본인의 마이페이지 > 기본환경설정에서 변경 할 수 있습니다. 설정값이 없다면 본사 정보가 노출 됩니다.
3, 사업자정보확인은 공정거래위원회 사이트에서 사업자등록번호 조회 후 주소창에 링크값을 복사해 입력하시면 됩니다.
※ 공정거래위원회 사이트 : http://ftc.go.kr/info/bizinfo/communicationList.jsp
<웹브라우져>
1, 웹브라우져 타이틀 : 익스플로러창 상단 좌측에 출력 됩니다. 하단 이미지 참조!
※ 참고, 1) 웹브라우져 타이틀은 가맹점몰과 본사몰이 각각 다르게 노출 됩니다.
※ 참고, 2) 단! 가맹점에서 값을 입력하지 않았다면 본사 설정값이 기본적으로 노출 됩니다.

2, 즐겨찾기시 멘트 : 즐겨찾기 버튼 클릭시 출력되는 멘트 창입니다. 하단 이미지 참조!
※ 참고, 1) 즐겨찾기시 멘트는 가맹점몰과 본사몰이 각각 다르게 노출 됩니다.
※ 참고, 2) 단! 가맹점에서 값을 입력하지 않았다면 본사 설정값이 기본적으로 노출 됩니다.

<회원가입>
1, 회원가입 포인트 : 신규회원 가입시 회원에게 지급되는 포인트 입니다. 미사용은 0을 입력하세요!
2, 추천인 포인트 : 신규회원을 추천한 회원에게 포인를 지급합니다. 미사용은 0을 입력하세요!

3, 보이기 : 회원가입시 필수 사항이 아닌 선택 입력사항으로 설정 됩니다.
4, 필수입력 : 회원가입시 반드시 입력해야만 회원가입이 됩니다.

5, 회원가입시 기본입력값 이외에 10개의 여분값을 설정 하실 수 있습니다. 예) 직장명, 직급등..
6, 여분필드값이 설정 되면 쇼핑몰, 회사홈페이지, 모바일 쇼핑몰에서도 모두 실행 됩니다.
7, 아이디 금지단어 : 회원가입시 설정 된 아이디로는 가입이 불가능 합니다. 공백없이 콤마로 구분 합니다.
8, 입력 금지 메일 : 회원가입시 설정 된 이메일은 입력 할 수 없습니다. 엔터로 구분 합니다.

<기타설정>
1, 접근가능 IP : 설정 된 IP만 쇼핑몰, 회사홈페이지, 모바일 쇼핑몰에 접속 할 수 있습니다. 엔터로 구분 합니다.
2, 접근차단 IP : 설정 된 IP는 쇼핑몰, 회사홈페이지, 모바일 쇼핑몰에 접속 할 수 없습니다. 엔터로 구분 합니다.
※ 주의) 접근차단 IP에 고객님의 IP주소를 절대로 입력하시면 안됩니다!
※ 참고) 123.123.+ 라고 입력하시면 IP주소 앞 2칸 123.123 이 속한 IP는 접속허용 및 차단 됩니다.

3, 찜바구니 보관일수 : 찜한 상품이 설정 기간동안 보관일이 경과 되면 자동으로 삭제 됩니다. 미사용은 0을 입력하세요!
4, 장바구니 보관일수 : 장바구니에 등록 된 상품이 설정 기간동안 보관일이 경과 되면 자동으로 삭제 됩니다. 미사용은 0을 입력하세요!
5, 구매결정 강제승인 : 배송완료 된 상품이 설정 기간동안 구매결정을 하지 않았을때 자동으로 구매결정 처리 됩니다. 미사용은 0을 입력하세요!
※ 참고) 공급사(위탁) 상품은 구매결정이 되지 않으면 정산처리를 하실 수 없습니다.

6, 재고관리 체크수량 : 상품관리 > 상품 재고관리에 노출 됩니다. 미사용은 0을 입력하세요!
7, 쿠폰 (온라인) : 온라인 쿠폰이란 상품구매 전 상품상세페이지에서 다운로드 후 주문시 즉시 사용가능한 쿠폰을 말합니다. 하단 이미지 참조!

8, 쿠폰 (인쇄용) : 인쇄용 쿠폰이란 쿠폰번호를 미리 난수로 생성 후 오프라인에서 배포합니다.
소비자는 그 쿠폰으로 쇼핑몰에 회원가입 후 사용기간내 쿠폰번호를 인증 받게 되면 쿠폰금액만큼 포인트로 전환받게 됩니다. 하단 이미지 참조!

<인트로설정>
1, 메인 인트로 : 복지몰, B2B몰, 폐쇄몰등으로 이용하실때 사용 됩니다. 하단 인트로 화면 모습

2, 회원인증 : 체크안하면 회원가입 완료후 바로 쇼핑몰 이용이 가능합니다.
※ 참고) 관리자페이지 > 회원관리에서 인증부분에 체크를 하셔야만 승인처리 됩니다. 하단 이미지 참조!

3, 가입 인증권한 : 각 가맹점에 소속 된 회원일경우 직접 인증할 수 있습니다. 가맹점관리자 > 회원목록에서 설정가능! 본사와 기능 동일!
