<판매자 정산관리>

※ 정산을 위해선 먼저 기간 검색을 하셔야 합니다. 위 이미지 참조!
<정산완료 전>

① 주문총계 : 판매가 + 옵션가 (배송비 제외!)
② 공급가총계 : 공급가 + 옵션가 (배송비 제외!)
③ 결제수수료 : 신용카드, 계좌이체, 휴대폰결제등 결제 수수료가 공급업체 부담이면 금액이 노출 됩니다.
④ 정산액총계 : 공급가총계 - 결제수수료
⑤ 가맹점수수료 : 현 업체 주문건중 지급된 가맹점 수수료 금액입니다.
⑥ 포인트적립 : 현 업체 주문건중 소비자회원에게 적립된 포인트금액 입니다.
⑦ 포인트결제 : 현 업체 주문건중 소비자회원이 주문시 결제한 포인트금액입니다.
⑧ 쿠폰할인 : 현 업체 주문건중 쿠폰사용으로 할인 된 금액 입니다.
⑨ 배송비 : 현 업체 주문건중 발생 된 배송비 금액 입니다.
⑩ 옵션가 : 공급가를 제외한 옵션가 금액입니다.
⑪ 실공급가 : 옵션가를 제외한 공급가 입니다.
⑫ 본사마진 계산법
　 a,결제수수료가 본사 부담일때 : (주문금액 - 결제수수료) - (가맹점수수료 + 적립금결제 + 포인트적립 + 쿠폰할인)
　 b,결제수수료가 업체 부담일때 : 주문금액 - (가맹점수수료 + 적립금결제 + 포인트적립 + 쿠폰할인)
⑬ 정산 완료 전에는 처리 내역이 없으므로 "0"으로 노출 됩니다.
⑭ "정산" 버튼을 클릭하시면 ④번 항목 "정산액총계" 금액이 정산 완료되며 ① ~ ⑫번 항목 모두 "0"으로 초기화 됩니다.
⑮ "업체명"을 클릭하시면 주문상세 내역을 모두 확인 하실 수 있습니다.

<정산완료 후>
