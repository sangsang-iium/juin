<상품 브랜드관리>
  • ① [브랜드명 (KOR)] : 필수 입력사항 입니다
  • ② [브랜드명 (ENG)] : 입력하지 않으셔도 무관합니다.
  • ③ [저장] : ①,②번 항목 모두 입력 후 "저장"버튼을 클릭하시면 브랜드가 신규생성 완료 됩니다.
  • ② 수정할 브랜드명을 입력하세요!
  • ① 적용할 항목에 체크 후 ⑤,⑥번 버튼을 클릭하시면 됩니다.
  • ③ [상품수정] : 버튼을 클릭시 "브랜드수정" 모드로 접속 합니다.
  • ④ [상품삭제] : 버튼을 클릭시 개별적으로 브랜드를 삭제 합니다.
  •  
  • ※ [브랜드 URL] : 링크주소를 복사 후 배너링크 주소에 입력하시면 해당 브랜드로 등록된 상품을 확인 하실 수 있습니다.
