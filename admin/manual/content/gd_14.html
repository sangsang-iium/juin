<상품 일괄수정시 주의사항>
  • 1, 컬럼(필드) 구분자에 쉼표( , ) 가 포함될 경우 error 가 발생되므로 모든 컬럼(필드) 에는 쉼표를 입력하지 않습니다.
  • 2, 엑셀에서 1번라인 컬럼(필드) 항목 타이틀 제목은 반드시 삭제하시고 업로드 하시기 바랍니다.
  • 3, 상품분류 입력시 " 0 "이 값이 없어지므로 가장앞에 #을입력하세요! 예시) #001;001001;001001001
<상품 일괄수정 절차 안내>
  • 1) 전제 상품관리에서 조회할 항목을 검색 후 엑셀저장
  • 2) 수정할 컬럼(필드)에 값을 변경 후 그대로 업로드 [컬럼(필드) 위치변경 불가합니다.]
<실제 상품 일괄수정 예시>
  • ① 예제다운로드를 하시고 각 컬럼에 맞는 데이터를 입력하세요!
  • ② 완성 된 파일을 첨부하시면 됩니다. 파일형식은 반드시 CSV여야 합니다.
  • ③ 등록하기 버튼을 통해 1차 업로드를 실행 합니다. 이때 실제 데이터가 DB에 저장되지는 않습니다.
  • ④ 업로드가 완료 되면 업로드 전 미리보기 버튼이 활성 됩니다. 클릭하시면 ⑤번 항목처럼 전체팝업이 실행 됩니다.
  • ⑤ 각 컬럼에 이상이없는지 확인 후 저장하기 버튼을 클릭 하시면 실제 데이터가 저장 됩니다.
