<게시판 목록>

② 게시판제목을 일괄/개별 수정시 ①번 체크박스를 선택 후 ⑥번 "선택수정" 버튼을 클릭 합니다.
③ 소속 그룹명 입니다.
④ 버튼을 클릭하시면 수정모드로 접속 됩니다.
⑤ 버튼을 클릭하시면 개별적으로 삭제 처리 됩니다.
　 ※ 일괄 삭제시에는 ①번 전체 체크및 선택 체크 후 ⑦번 "선택삭제" 버튼을 클릭 합니다.
⑦ 게시판을 신규생성 합니다.
　 ※ 기존 게시판 TABLE 번호 22 ~ 40번까지는 삭제 하실 수 없습니다. DB강제 삭제시 오류 발생 됩니다.

<게시판 생성 및 수정>
[TABLE] : 한번 생성 된 TABLE 번호는 수정하실 수 없습니다.
[그룹] : 게시판 소속 그룹 지정 (그룹 지정에 따라 노출 위치가 달랍니다.)
　 ※ 쇼핑몰 : 쇼핑몰에 노출
　 ※ 공급업체 : 공급사 마이페이지에 노출
　 ※ 홈페이지 : 회사 홈페이지에 노출
[게시판 제목] : 원하시는 게시판제목을 입력하세요.
[스킨 디렉토리] : 게시판 스킨을 설정 합니다.
　 ※ basic : 기본 게시판
[분류] : 분류와 분류 사이는 공백없이 | 로 구분하세요. (예: 질문|답변) 첫자로 #은 입력하지 마세요. (예: #질문|#답변 [X])
[게시판 테이블 폭] : 게시판 가로 넓이 크기 입니다.
[페이지당 목록 수] : 게시판에 노출 될 게시물 총 라인수
[제목 길이] : 설정된 값만큼 글자가 노출 됩니다. 예시) 행복을 주는 쇼핑....
[상단 파일 경로] : 게시판 상단에 작업 된 페이지 경로를 설정 합니다.
[하단 파일 경로] : 게시판 하단에 작업 된 페이지 경로를 설정 합니다.
[상단 이미지] : 게시판 상단 좌측에 제목으로 노출 됩니다. 너무 큰 이미지는 권장하지 않습니다.
[하단 이미지] : 게시판 하단에 이미지를 노출 합니다.
[상단 내용] : 게시판 상단에 특정 문구나 HTML을 노출 합니다.
[하단 내용] : 게시판 하단에 특정 문구나 HTML을 노출 합니다.
[글쓰기 기본 내용] : 글쓰기 작성 폼안에 기본 내용이 노출 됩니다. 예시) 이름 : 나이 : 등등..
[목록보기 권한] : 게시판 목록에 접근 할 수 있는 권합니다. 권한지 맞지 않으면 접근 자체가 안됩니다.
[글읽기 권한] : 게시판 작성글에 접근 할 수 있는 권합니다. 권한지 맞지 않으면 접근 자체가 안됩니다.
[글쓰기 권한] : 게시판 글쓰기 권한 입니다.권한지 맞지 않으면 글작성을 할 수 없습니다.
[글답변 권한] : 글 작성내용에 답변 할 수 있는 권한입니다. 권한지 맞지 않으면 작성 할 수 없습니다.
[코멘트쓰기 권한] : 글 작성내용 하단에 댓글 작성 쓰기 권한입니다. 권한지 맞지 않으면 작성 할 수 없습니다.
[파일 업로드] : 기본적으로 파일을 2개까지 업로드 가능합니다. 체크하지 않으면 히든 처리 됩니다.
[비밀글 사용] : 작성자와 관리자외에는 게시글을 확인 할 수 없습니다.
　 ※ 체크박스 : 작성자의 선택에 따라 비밀글로 설정 가능 합니다.
　 ※ 무조건 : 작성자가 선택할 수 없으며 작성 된 글은 무조건 비밀글로 처리 됩니다.
[글내용 옵션]
　 ※ 전체목록 : 게시물 하단에 전체 리스트가 출력 됩니다.
　 ※ 이전글 다음글만 사용 : 게시물 하단에 전체 리스트가 노출 되지 않고 이전글 다음글만 노출 됩니다.
