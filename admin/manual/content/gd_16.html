<쿠폰관리 (온라인)>
  • [쿠폰발급 유형]
    1) 발행 날짜 지정 : (발급주기 : 자유) 설정해놓은 기간에만 쿠폰이 발급됩니다.
    예) 2014년 12월 29일부터 2015년 02월 29일까지
    2) 발행 시간/요일 지정 : (발급주기 : 자유) 설정해놓은 요일과 시간대별로 발급됩니다.
    예) 월,수,금 오후1시부터 오후 3시까지 10매 발급
    3) 성별구분으로 발급 : (발급주기 : 자유) 남성, 여성, 모든고객으로 구분하여 발급됩니다.
    예) 2014년 12월 29일부터 2015년 02월 29일까지 여성고객에게만 발급
    4) 회원 생일자 발급 : (발급주기 : 1년에 한번) 회원 생일에 발급되며 1년에 단 한번만 발급됩니다.
    예) 행복하고 즐거운 생일쿠폰 발급
    5) 연령 구분으로 발급 : (발급주기 : 자유) ○○○○년생부터 ○○○○년생까지 발급됩니다.
    예) 2009년생부터 2014년생까지 발급
    6) 신규회원가입 발급 : (발급주기 : 최초 1회) 회원가입시 최초1회 발급
  • [누적제한] : 최대 ○○건 까지만 다운로드 가능! (무제한 : "99999999999")
  • [레벨제한]
    1) "제한없음"일 경우 회원이면 누구나 다운로드 가능합니다.
    2) 특정레벨 선택시 선택된 회원등급 이상만 다운로드 가능합니다.
<필수입력사항>
  • [쿠폰명] : 실제 보여질 쿠폰명을 입력하세요!
  • [설명] : 쿠폰에 대한 간략한 설명을 입력하세요! 쿠폰명과 동일해도 무관하며 입력하지 않으셔도 됩니다.
  • [사용여부] : 사용중지로 설정하시면 쇼핑몰에 노출되지 않습니다.
  • [쿠폰발급 시점]
    1) 회원직접 다운로드 : 상품상세정보에서 회원이 직접 쿠폰을 다운로드받습니다.
    2) 주문완료 후 자동발급 : 주문완료 즉시 쿠폰을 발급합니다. 입금대기 상태에서도 고객이 쿠폰을 받을 수 있습니다.
    3) 주문완료 후 배송완료시에 자동발급 : 주문상태가 배송완료로 변경되는 시점에 쿠폰을 발급합니다.
  • [중복발급 여부]
    1) 중복발급 안함 : 한 고객에게 무조건 1회만 쿠폰을 발급합니다.
    2) 중복발급 허용 : 쿠폰을 사용한 후 다음번 주문시에도 같은 상품의 쿠폰다운로드를 허용합니다.
  • [해택]
    1) (상품 판매가 x 수량) 의 ○% 할인, 최대 ○○○원까지만 할인적용 됩니다.
    2) ○○○원할인, 입력값 그대로 할인 됩니다.
  • [중복사용 여부]
    1) 이 쿠폰은 동일한 주문건에 중복해서 동시에 사용할 수 있습니다. (동일한 주문건에 여러번 사용가능합니다)
    2) 이 쿠폰은 동일한 주문건에 중복해서 동시에 사용할 수 없습니다. (동일한 주문건에 한번만 사용가능합니다)
<필독하세요!!>

  • ※ 예를들어 쿠폰명을 수정하였거나 쿠폰 발급 기간을 수정했더라도 이미 발급된 쿠폰에는 반영되지 않습니다.
  • ※ 쿠폰을 삭제 했더라도 기존에 발급된 쿠폰은 삭제 되지 않습니다.
  • ※ 고객은 발급 당시 쿠폰설정 그대로 사용 됩니다.
