[상품 주문옵션]
※ 주문조건 : 소비자회원이 주문옵션을 반드시 선택해야 주문이 가능한 필수항목 입니다.
※ 용도 : 본 상품에 대한 다양한 타입 및 가격 설정 예시) 티셔츠 타입 A, 타입 B, 타입 C
※ 구매수량 : 구매수량 데이터에 반영 됩니다. 주문금액 계산 : (본 상품금액 + 추가금액) × 수량
※ 부분취소 여부 : 주문옵션별 부분취소/반품이 가능 합니다.

① 옵션1,2,3 항목에 콤마로 구분하여 옵션 소분류를 등록한 후 ②번 "옵션목록생성" 버튼을 클릭합니다.
③ 옵션이 자동으로 생성되어 배열 됩니다.
④ 전체 옵션의 추가금액, 재고/통보수량 및 사용여부를 일괄 적용할 수 있습니다. 단, 체크된 항목만 적용됩니다.
　 ※ 재고수량 무제한 : 상품주문시 재고가 차감 되지 않습니다.
　 ※ 통보수량 무제한 : 상품관리 > 상품 옵션재고관리에 노출 되지 않습니다.

⑤ "옵션일괄적용" 버튼을 클릭하시면 ④번에 체크 된 항목만 일괄적으로 적용 됩니다.
[상품 추가옵션]
※ 주문조건 : 소비자회원이 추가옵션을 선택하지 않아도 주문이 가능한 선택항목 입니다.
※ 용도 : 본 상품의 액세서리 및 코디상품을 ‘추가’ 판매 예시) 카메라 삼각대, 케이스, 메모리카드
※ 구매수량 : 구매수량 데이터에 반영되지 않습니다. 주문금액 계산 : (본 상품금액 × 수량) + (추가구성 금액 × 수량)
※ 부분취소 여부 : 주문옵션별 부분취소/반품이 불가 합니다.

※ "옵션추가" 버튼을 클릭하시면 등록 가능한 옵션 항목이 무한 생성 됩니다.
※ 이하 작동 원리는 위 주문옵션과 동일 합니다.

<상품 가격정보>

[시중가격] : 시중에서 판매되고 있는 가격을 입력하세요.
[판매가격] : 쇼핑몰에 실제 판매할 가격을 입력합니다. 상품 등록 시 필수항목입니다.
[가격 대체문구] : 상품가격이 정해져 있지 않고, 전화상담을 받아 처리를 하고자 할 경우 사용합니다.
　 ※ 전화상담 이라고 입력 하시면 해당 상품은 판매가격 대신 전화상담으로 표시되고 상품 주문을 할 수 없으며,
　　  구매하기, 장바구니, 찜, 대량구매 버튼과 옵션도 노출되지 않습니다.
[수량]
　 ※ 무제한 : 재고수와 상관없이 무제한 주문이 가능합니다. 주문시 수량 감소 안됨!
　 ※ 한정 : 재고수를 체크하며 재고 부족시 주문이 불가능 합니다. 주문시 수량 감소!
　 ※ 재고 통보수량 : 상품의 재고가 통보수량보다 작을 때 상품 재고관리에 표시됩니다.
　　  단! 옵션이 등록 된 상품은 옵션의 통보수량이 적용됩니다. 옵션의 재고수량,통보수량이 무제한이면 옵션재고관리에 표시되지 않습니다.
[주문한도]
　 ※ 최소 : 1회 주문 시 최소주문수량을 입력합니다. 한번에 여러개를 판매해야 하는 상품일 경우 입력을 해주세요.
　 ※ 최대 : 최소주문수량과는 반대로 1회 주문 시 가능한 최대 주문량을 입력합니다.
　 ※ 최소, 최대 둘중 하나만 입력도 가능하며 최소에 값이 있을 경우 수량에 기본적으로 고정되어 표시 됩니다. 이하 주문수량은 주문 불가!

[적립금] : 주문이 배송완료시 소비자 회원에게 적립 될 금액을 입력하세요.
　 ※ ○원 부분에 입력하시면 금액 그대로 저장되며 ○%로 입력하시면 판매를 기준으로 자동계산 되어 적용 됩니다.
[판매기간 설정] : 설정된 기간 동안만 판매 가능하며, 설정된 종료일 이후에는 판매되지 않습니다.
　 ※ 일시 판매중지 처리하실 경우, 종료일을 현재날짜 이전의 과거 날짜를 넣어주시면 됩니다.
　 ※ 초기화 부분에 체크하시면 설정 된 날짜가 모두 초기화 됩니다.

[구매가능 레벨] : "제한없음"이면 비회원도 모두 구매 가능하며, 선택 된 레벨 이상만 구매 가능합니다.
　 ※ 현재 레벨이상 가격공개에 체크시 레벨 조건에 맞지 않으면 시중가, 판매가, 적립금, 옵션이 모두 노출되지 않습니다.
　 ※ 구매가능 레벨을 일반회원으로 설정 하셨다면 비회원의 경우 판매가가 회원공개 아이콘이 노출되며,
　　  시중가, 적립금, 옵션이 모두 노출되지 않습니다. 로그인 후에는 모두 정상 노출되고 구매 가능합니다.

[수수료 적용타입] : 상품등록시 기본값은 "공통설정" 이며, 가맹점관리 > 가맹점 수수료정책에서 변경 합니다. 매뉴얼 참조!

※ 개별설정으로 적용 시 (원/%) 중 원하시는 타입을 선택하시고 단계란에 숫자를 입력하시면 금액을 입력 할 입력폼이 활성 됩니다.
