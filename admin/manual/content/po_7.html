<반드시 숙지하세요!>

① 관리자페이지 가맹점관리 > 가맹점 환경설정에서 수수료 정산일자가 월단위, 주단위 수수료정산일 경우 노출 됩니다.
② 관리자페이지 가맹점관리 > 가맹점 환경설정에서 수수료 정산일자가 실시간 수수료정산일 경우 노출 됩니다.

<월단위 수수료정산 사용시 : 기간검색>

① 월단위 검색은 년/월 형식으로 검색 됩니다. 수수료 정산시 지급할 기간을 먼저 조회를 하셔야 합니다.
② 월단위 검색시 수수료가 적립 된 해당 월 기간이 노출 됩니다.
※ 월단위 사용시 주별검색은 비활성되며 주별 검색은 하실 수 없습니다. 년/월만 검색 가능합니다.

<주단위 수수료정산 사용시 : 기간검색>

① 주단위 검색은 해당 월별로 매주 (일요일~토요일)까지 7일 단위로 검색 됩니다. 수수료 정산시 지급할 기간을 먼저 조회를 하셔야 합니다.
② 주단위 검색시 수수료가 적립 된 해당 주 기간이 노출 됩니다.
※ 주단위 사용시 주별검색이 활성되며 년/월/주별 검색 모두 사용 가능합니다.

<수수료 정산하기>

① 수수료 정산을 하시고자 하는 항목에 체크 후 ⑨번 정산완료 버튼을 클릭하시면 수수료가 정산처리 됩니다.
　 ※ 수수료 정산 순서
　 a, ⑧번 엑셀 다운로드 후 자주 사용하신 인터넷뱅킹 대량이체 CSV형식으로 변경합니다. 예시) 은행명, 계좌번호, 예금주명, 이체금액
　 b, 대량 이체를 완료 하셨다면 위 항목처럼 정산완료 처리 하시면 됩니다.
② 세금공제 전 실제 수수료 잔액 입니다.
③ 세금공제는 관리자페이지 가맹점관리 > 가맹점 환경설정에서 설정 된 퍼센트(%)만큼 공제 됩니다.
④ 수수료 잔액에서 세금공제를 제외한 실제 정산 될 금액입니다.
⑤ 가맹점 회원에게 정산 될 은행계좌 정보 입니다.
　 ※ 은행계좌정보 변경 안내
　 a, 가맹점이 직접 계좌정보를 변경시 가맹점 마이페이지 > 기본환경 설정에서 변경 가능합니다.
　 b, 관리자가 직접 계좌정보를 변경시 회원명을 클릭! 회원정보수정 팝업창에서 변경 가능합니다. 매뉴얼 회원관리 > 회원 정보관리 참조!
⑥ 수수료가 적립 된 기간입니다.
⑦ 정산이 완료 되기 전 상태 입니다.
⑩ 정산을 유보하실 때 ①번 항목에 체크 후 클릭 하시면 됩니다.
⑪ 정산을 거절하실 때 ①번 항목에 체크 후 클릭 하시면 됩니다.

<참고사항>
※ 정산처리가 완료 되면 현재 페이지에서 노출되지 않고 가맹점 정산처리내역으로 이동 됩니다.
※ ⑥번 적립기간이 다르면 같은 회원이라도 중복해서 노출 될 수 있으니 반드시 정산하기 전 기간을 먼저 검색하시기 바랍니다.
※ 본 페이지에서는 모든 가맹점이 노출되는 곳이 아닙니다. 정산할 금액있는 회원만 노출 됩니다.
